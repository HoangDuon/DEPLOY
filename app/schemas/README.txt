Thư mục định nghĩa các schema (Pydantic models) cho request và response API.

Các schema giúp kiểm tra dữ liệu đầu vào (validation) và định dạng dữ liệu trả về.

Ví dụ:
- user_schema.py: UserCreate, UserRead, UserUpdate
- class_schema.py: ClassCreate, ClassResponse
- ticket_schema.py: TicketCreate, TicketResponse, ...

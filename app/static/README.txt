Thư mục chứa các tài nguyên tĩnh (static files) phục vụ giao diện hoặc nội dung nội bộ.

Bao gồm:
- css/: stylesheet
- js/: script dùng trong giao diện HTML (nếu render từ backend)
- images/: hình ảnh, logo, icon nội bộ

⚠️ Nếu frontend đã tách riêng (VD: React/Vue), thư mục này có thể chỉ dùng để phục vụ file tĩnh nhỏ.
